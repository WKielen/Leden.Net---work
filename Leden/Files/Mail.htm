Selectie van leden die moeten worden gemaild

De leden die moeten worden gemaild kunnen worden geselecteerd via het "Selectie" scherm via de "Select"-knop.

Er is ook een mogelijkheid om snelle selecties te maken. Eerst moet linker vak aanklikken in het Mail scherm. Daarna kunnen sneltoesten worden gebruikt. De sneltoesten zijn:

A = Alle leden
J = Jeugd
Ctrl-J = Jeugd + Sen1
S = Senior

C = Alle competitie gerechtigde leden
Shift C = Senior competitie gerechtigd
Ctrl-C = Jeugd competitie gerechtigd

N = Alle leden met Bondsnummer
Shift N = Senior met Bondsnummer
Ctrl-N = Jeugd met Bondsnummer
 
Ctrl-A = Jeugd A
Ctrl-B = Jeugd B
Ctrl-R = Jeugd Ranglijst speler
Ctrl-T = Jeugd Toernooi speler
 
V =Vrijwilligers met Vaste Taak (alleen senior)
Shift V = Leden die taak moeten uitvoeren

Copy/Paste vanuit Word
Gebruik de "Paste" knop als je tekst vanuit Microsoft Word kopieert. De overbodige control-tekens worden dan verwijderd.
